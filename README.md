# OnlineMastering (local demo)

Минимальный локальный демо‑скрипт автоматического мастеринга: берёт входной WAV и пишет “мастер” WAV.

## Установка

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python master.py input.wav output.wav
```

## Что делает скрипт (коротко)

- Мягко создаёт запас по уровню, если вход уже “в клиппинге”
- Лёгкая коррекция низа/верха (shelf EQ) по грубому спектральному анализу
- Очень мягкая компрессия
- Лимитирование и нормализация по LUFS (цель выбирается эвристикой: примерно -14 / -12 / -10 LUFS)
